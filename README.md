# 🦷 Projeto Sorriso Perfeito - Landing Page

---

## 📌 Sobre o Projeto

A **Sorriso Perfeito** é uma clínica odontológica fictícia desenvolvida como projeto acadêmico com o objetivo de criar uma **landing page profissional, moderna e responsiva**.

Este projeto reforça conhecimentos em:

- HTML5 (estrutura semântica)
- CSS3 (estilização e responsividade)
- JavaScript (interatividade e validação)
- Organização modular de arquivos
- Preparação para futura integração com PHP

A identidade visual transmite **limpeza, cuidado, confiança e modernidade**, características essenciais para um consultório odontológico.

---

## 🎯 Objetivo

Criar uma landing page responsiva para o consultório fictício **Sorriso Perfeito**, contendo:

- Apresentação institucional
- Serviços oferecidos
- Galeria de sorrisos
- Depoimentos de pacientes fictícios
- Formulário de contato validado com JavaScript

## ▶ Como Executar o Projeto

1. Clone o repositório:
git clone https://github.com/seu_usuario/projeto-sorriso-perfeito.git


2. Acesse a pasta do projeto:
cd projeto-sorriso-perfeito


3. Abra o arquivo `index.html` no navegador.

Dica: Utilize a extensão **Live Server** no VS Code para melhor experiência.
